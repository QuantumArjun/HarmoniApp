# Harmoni
A better way to engage your employees

## Running Harmoni
- Clone this repo
- `npm install`
- `npm start`

## Adding a New Feature

- `git checkout master`
- `git pull`
- `git checkout -b <NEW-FEATURE-NAME>`
-  Modify the Files
- `git add <NEW-FILES>`
- `git commit -m <MESSAGE>`
- `git push origin <NEW-FEATURE-NAME>`
